# MELLOWTREMOR POC
Content filtering technologies are sometimes vulnerable to filtration bypass with Google Translate. 
The site acts as a proxy, displaying the content of the selected site on the blocklist inside an iFrame within Google Translate itself, thus creating the bypass.
